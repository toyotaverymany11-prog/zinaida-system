## КЛЮЧЕВАЯ СТРАТЕГИЯ: ОДИН КАНАЛ + БОТ-СЕГМЕНТАЦИЯ

Не 6 каналов под 6 фаз — а **один канал Telegram + бот**, который сегментирует аудиторию.

Схема: **Instagram/VK (прогрев) → Telegram (канал + бот) → MessengerMax (закрытый контент) → Продукт**

### Принципы
1. **Думать от боли, а не от категории.** Девушка не знает свою «фазу». Она знает «он не пишет», «изменил», «хочу замуж».
2. **Контент не делится на фазы публично.** Каждый пост цепляет универсальную боль + лид-магнит в Telegram.
3. **Разный контент под каждую соцсеть.** Не копипастить. Instagram ≠ VK ≠ Дзен.
4. **Собирать аудиторию в Telegram.** Это единственная платформа, где аудитория твоя, а не алгоритма.

## ВОРОНКА 2026

```
Instagram/VK (прогрев) → Telegram (конверсия) → Продукт (продажа)
```

**Топ: Instagram/VK** — карусели 10 слайдов, сторис с опросами, болевые кейсы. 3-5 постов/нед. Цель: вовлечение → лид-магнит.

**Середина: Telegram** — автоворонка через бота. Приветствие → вопрос про фазу → чек-лист → серия писем → трипвайер 99₽.

**Низ: MessengerMax** — дублирование воронки для тех, кто не любит Telegram.

## МЕТРИКИ И KPI

**Контентные метрики (пост):**
- Saves Reach Ratio: сохранения / охват
- Share Rate: пересылки / охват  
- Completion Rate: дочитываемость
- Conversion CTR: переход по ссылке

**Бизнес-метрики:**
- Подписчики Telegram: рост/неделю
- Конверсия в бота: % перешедших по ссылке
- Конверсия в трипвайер (99₽): % от подписчиков
- Конверсия в основной продукт: % от трипвайера
- Unit-экономика: себестоимость 1 поста ≈ $4.60 (LLM $3.00 + визуал $1.50)

## РИСКИ ДЛЯ МАРКЕТИНГА

| Риск | Описание | Митигация |
|------|----------|-----------|
| **Теневой бан** | Контент невидим без причин | Не нарушать правила платформ, диверсификация |
| **Алгоритмическая зависимость** | Изменение алгоритмов убивает охват | Собирать базу в Telegram |
| **Госнадзор (152-ФЗ)** | Цензура медицинских тем | Дисклеймеры, algospeak |
| **AI-модерация** | Автоблокировка по ключевым словам | Замена: измена → «смотрит налево» |
| **Зависимость от одного канала** | Если канал блокируют — всё | VK + Telegram + MessengerMax |

## МОНЕТИЗАЦИЯ

Воронка продаж:
1. **Бесплатный контент** — посты в VK/Instagram (прогрев)
2. **Чек-лист/гайд** — бесплатный лид-магнит в Telegram (конверсия)
3. **Трипвайер 99₽** — тест/разбор ситуации (первая продажа)
4. **Основной продукт** — подписка на Зинаиду, курс, консультации

**Автоворонка через бота Telegram:**
Приветствие → вопрос про фазу отношений → бесплатный чек-лист под её фазу → серия 3-5 писем → трипвайер 99₽ → основной продукт
