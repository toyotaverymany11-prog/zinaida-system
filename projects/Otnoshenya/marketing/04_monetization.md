## 2. ЛИД-МАГНИТ (СЕРЕДИНА ВОРОНКИ)

**Что:** бесплатный чек-лист/гайд под фазу отношений
**Где:** Telegram-бот выдаёт после сегментации
**Цель:** конверсия из подписчика в лид

**Варианты лид-магнитов:**
- «5 признаков, что он не вернётся» — для фазы тревоги
- «Как говорить, чтобы он слышал» — для фазы избегания
- «Тест на психотип вашего мужчины» — универсальный
- «3 фразы, которые вернут его из молчания» — для фазы гостинга

## АВТОМАТИЗАЦИЯ ПРОДАЖ

Вся воронка автоматизирована через Telegram-бота:

1. Пользователь подписывается на канал
2. Бот: «Привет! Я Зинаида. Чтобы я могла давать тебе релевантный контент — скажи, где ты сейчас?»
3. Выбор фазы → бесплатный чек-лист
4. Через 3 дня → серия писем с углублением
5. На 7-й день → предложение трипвайера 99₽
6. После покупки → прогрев до основного продукта
7. Через 14 дней → предложение основного продукта
