# ТАКТИКИ ПО ПЛАТФОРМАМ — ЧТО И КАК ПУБЛИКОВАТЬ

## VK (ВКОНТАКТЕ)
**Группа:** aipsiholog | **Страница:** zinadchdp | **Статус:** открыта, 0 постов

### Что работает
- **Длинные посты** 2000-5000 знаков. VK любит текст, не короткие ленты.
- **Истории из жизни** — переформатированные микро-истории из phases.db
- **Разборы** — научный факт + бытовой перевод + вывод
- **Опросы** — увеличивают вовлечение, влияют на алгоритм
- **Обсуждения** — 3 уже созданы, нужны ответы

### Что не работает
- Короткие посты (как в Instagram)
- Посты без картинки (тонут в ленте)
- Перепосты из других соцсетей

### Формат поста
```
[Цифра/факт в первой строке]
[Картинка 1080×1080]
[Текст 2000-5000 зн. — разбор + история + вывод]
[Опрос/вопрос в конце]
[Ссылка на Telegram: «Полный разбор — в канале»]
```

### Расписание
Лучшее время: будни 19:00-23:00, выходные днём.
Частота: 3-5 постов/неделю.
Хештеги: 3-5, релевантные.

### API
wall.post, photos.getWallUploadServer.
Нужен ключ сообщества (Олег: Управление группой → Работа с API → Создать ключ).

## INSTAGRAM
**Статус:** не открыта

### Что работает
- **Карусели** (10 слайдов) — лучший формат для Saves
- **Reels** — единственный способ получить охват без подписчиков
- **Storis с опросами** — вовлечение

### Что не работает
- Текст без визуала
- Длинные тексты (лимит 2200 зн.)
- Хештеги-спам (3-5 релевантных, не больше)

### Формат карусели
```
1 слайд: Хук (цифра/факт + шокирующий заголовок)
2-3 слайд: Проблема (микро-история, узнавание)
4-6 слайд: Разбор (научный факт → бытовой перевод)
7-8 слайд: Решение (конкретные шаги)
9 слайд: Вывод (жёсткая правда)
10 слайд: CTA (ссылка на Telegram)
```

### Reels
- 15-60 сек
- Первые 3 секунды — самый сильный хук
- Текст на экране (субтитры обязательны)
- Музыка из трендов

### Расписание
Лучшее время: 12:00-14:00, 20:00-22:00.
Частота: 3-5 постов + 3-5 сторис/неделю + 2-3 Reels.

## АНАЛИЗ КОНКУРЕНТНОЙ СРЕДЫ (РФ)

Рынок онлайн-рекламы: $6.5 млрд.
Лидирующие платформы: VK, Telegram, Яndex.

### Конкуренты в нише психологии отношений
- **Женские паблики VK:** огромный охват, но низкое качество
- **Instagram-психологи:** высокая конкуренция, много повторов
- **Telegram-каналы:** растут, но ниша «мужская психология» слабо занята
- **Дзен:** хороший трафик из поиска

### Наше преимущество
- **AI-аналитика + человеческий голос Зинаиды.** Единственный проект, который сочетает большие данные и бытовую подачу.
- **Системность.** Не случайные посты, а 7-слойная архитектура.
- **Ниша «мужская психология»** — конкретная, не размытая «отношения вообще».
