# Планировщик — система управления задачами и памятью

## Назначение
Полноценный активный планировщик на Todoist + Telegram + Hermes.
Ведёт все задачи Олега: проекты, встречи, покупки, стратегические и текущие дела.

## Принцип работы
- В любом чате: «запомни/создай задачу/напомни» → я создаю в Todoist
- Утром (7:00) — дайджест в Telegram: выполнено, в работе, сегодня, новое
- Проверяю статусы в течение дня, напоминаю о дедлайнах
- Раз в неделю — ревью всех проектов

## Связанные компоненты
- Todoist API — токен в /opt/zinaida/meta_agent/.env
- Telegram notify — /opt/zinaida/telegram_bot/notify.py
- CronJob Hermes — 7:00 дайджест (id: 4646987b527f)
- Навык todoist-planner — /root/.hermes/skills/productivity/todoist-planner/

## Структура папки
- 00_README.md — этот файл
- config.md — настройки и ключи
- tasks/ — текущие задачи и статусы
- notes/ — заметки по планированию
