# ОЦЕНКИ ОЛЕГА ПО ДИЗАЙНУ
## Проект «Отношения»

### 09.07.2026 — Кожа и крупные планы
- **Тест кожи (test_skin.jpg):** ❌ Не нравится
- **Причина:** Слишком рисованная, ненатуральная, кожа слишком ровная
- **Решение:** Фото на отдалении (по пояс), не крупный план

### 09.07.2026 — Общий вывод по фото
- Крупный план лица — заметна неестественность кожи
- Нужно: отдаление, full body / half body
- Кожа должна выглядеть как настоящая, с imperfections

### 09.07.2026 — Шрифты
- Шрифт на референсе — serif (Didone / Playfair Display)
- НЕ гротеск, НЕ sans-serif для заголовков
- Высокий контраст штрихов
- Элегантный, премиальный

### 09.07.2026 — Композиция
- На референсе девушка ПЕРЕД буквами, буквы как рамка
- Белый фон, не тёмный
- Крупный текст сверху и снизу
- Текст должен читаться чётко — не сливаться с фоном

### 10.07.2026 — Язык
- Писать «Его страх» по-русски. Не «Ego Страх».

### 10.07.2026 — Дизайн-процесс
- ЗАПРЕЩЕНО накладывать текст через Python/Pillow
- Только через Леру (Replicate API + LoRA)
- Три варианта сразу, параллельно
- Перед показом — проверить глазами

### 10.07.2026 — Оценка стиля
- Didone / Modern Serif — идеально для ЦА (женщины 18-40)
- Считывается как «дорогой журнал», экспертность
- Это шрифт ВЛАСТИ и ЗНАНИЯ, не «девочковый»
