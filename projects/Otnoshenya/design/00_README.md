### ⚠️ ЖЕЛЕЗНЫЕ ПРАВИЛА ДИЗАЙНА (Олег)

1. **Дизайн — только через Леру.** Не через Python/Pillow/ImageMagick. Скрипты — только для техобработки (ресайз, обрезка).
2. **Три варианта сразу.** Не итерациями по одному. Параллельно.
3. **Перед показом Олегу — проверить через vision_analyze.** Не говорить «ок» если не ок.
4. **Всё фиксировать** — и хорошее, и плохое.
5. **Писать «Его страх» по-русски.** Не «Ego Страх».
7. **Все глаголы — женского рода.** Сделала, проверила, пошла.
8. **Без английского в ответах** (кроме техтерминов).
9. **Контекст в задании Лере.** Перед заданием — прочитать реестр. ВСЕ параметры вписать в задание (шрифт, цвета, референс, размер, что НЕЛЬЗЯ). Задание — самодостаточно, не ссылками.
10. **Сначала ОДИН тест, потом батч.** Перед любой серийной генерацией — сделать ОДИН тестовый образец. Проверить через vision_analyze. Только если он прошёл — запускать остальные. Не жечь деньги на 9 штук пока не убедились что промпт работает.

### РЕФЕРЕНС СТИЛЯ — IMG_1288.jpeg («Его страх»)

- Ч/б, высокий контраст
- Белый/чистый фон (НЕ тёмный)
- Шрифт с засечками: Didone / Bodoni / Playfair Display
- Крупный текст сверху и снизу (текст как рамка)
- Лицо ПЕРЕД буквами (часть букв за головой)
- Прямой дерзкий взгляд в камеру
- Плечи видны (half body, НЕ крупный план)

**Для обложки:** «МЕГАМОЗГ» наверху, «AI-аналитик мужской психологии» внизу.

### ТРИ НАПРАВЛЕНИЯ, КОТОРЫЕ ТЕСТИРУЕМ

| # | Направление | Модели | Цена | Риски |
|---|-------------|--------|------|-------|
| A | Seedream 5.0 + 3-4 reference фото | bytedance/seedream-5-lite | $0.035 | Лицо может не скопироваться |
| B | FLUX+LoRA (портрет) + FLUX Fill (текст) | flux-dev + toyotaverymany11-prog/zina + flux-1-fill-dev | $0.05 | Сложнее, 3 шага |
| C | Recraft V3 повторно | recraft-ai/recraft-v3 | $0.04 | Лицо не Зинаиды |

**Моя рекомендация:** Направление B (FLUX+LoRA+Fill) — лицо Зинаиды гарантировано через LoRA, текст через Fill становится частью картинки.
