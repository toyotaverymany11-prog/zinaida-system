## ⚠️ ЖЕЛЕЗНЫЕ ПРАВИЛА ДИЗАЙНА

### 1. ДИЗАЙН — ТОЛЬКО ЧЕРЕЗ ЛЕРУ
**ЗАПРЕЩЕНО** генерировать дизайн/картинки/наложение текста самостоятельно через Python/Pillow.
Зинаида: (1) даёт задание Лере через сервис, (2) контролирует результат, (3) показывает Олегу.
Скрипты Зинаиды — только для технической обработки (обрезка, ресайз, конвертация). НЕ творчество.

### 2. ВСЁ ФИКСИРОВАТЬ: и хорошее, и плохое
Каждый тест, каждая ошибка, каждый успех — в этот реестр. Чтобы система не деградировала, а улучшалась.

### 3. ПЕРЕД ПОКАЗОМ — ПРОВЕРИТЬ
Не отправлять Олегу непроверенный результат. Открыть, посмотреть глазами, убедиться что текст читается, лицо на месте, нет косяков.

### 4. ТРИ ВАРИАНТА СРАЗУ
Не делать 10 итераций по одной. Сразу 3 параллельных подхода → Олег выбирает → дорабатываем.

## ТРИ НАПРАВЛЕНИЯ ДЛЯ ТЕСТИРОВАНИЯ

### НАПРАВЛЕНИЕ A: Seedream 5.0 Lite + Reference (Текст)
**Цель:** Получить кириллицу + лицо Зинаиды в одной генерации.
**Модель:** bytedance/seedream-5-lite
**Фишки:**
- Текст в двойных кавычках "МЕГАМОЗГ" рендерится точно
- До 14 reference images для лица
- 2K разрешение (2048×2048)
- Цена: ~$0.035

**Что тестировать:**
1. Одно reference фото (10_portrait_v3.jpg) — не сработало
2. **Несколько фото (3-4 из assets/photos)** — может улучшит сходство
3. Промпт: "same face as reference image, identical person" — усилить

### НАПРАВЛЕНИЕ B: FLUX Dev + LoRA + Inpainting (Гибрид)
**Цель:** Лицо через LoRA + текст через FLUX Fill.
**Модели:** 
- Портрет: black-forest-labs/flux-dev + toyotaverymany11-prog/zina (LoRA)
- Текст в картинку: black-forest-labs/flux-1-fill-dev (Inpainting)

**Процесс:**
1. FLUX + LoRA → портрет Зинаиды без текста ✅ (умеем)
2. Наложить текст через скрипты (Pillow) с маской
3. FLUX Fill → дорисовывает свет/тени на текст → становится частью картинки
4. Цена: $0.025 + $0.025 = $0.05 за финалку

**Что нужно сделать:**
1. Найти LoRA `Muapi/russian-text-flux` на Civitai (для усиления текста, если нужно)
2. Настроить FLUX Fill с маской
3. Протестировать inpainting

### НАПРАВЛЕНИЕ C: Recraft V3 (Стильный дизайн)
**Цель:** Получить графический дизайн, а не фотореализм.
**Модель:** recraft-ai/recraft-v3
**Фишки:**
- Лучшая кириллица по данным форумов
- Стиль: вектор/иллюстрация/дизайн (не фото)
- Цена: $0.04

**Что тестировать:**
1. Другой промпт (предыдущий был неудачный)
2. Style: realistic_image / digital_illustration
3. Сравнить стиль с референсом

### Три типа контента

**Тип 1 — Журнальная обложка (60%)**
- Крупный Didone-шрифт (70% площади)
- Зинаида затемнённая на фоне (30%)
- Ч/б + один акцент (бордовый/золото)
- Размер: 1080×1080
- Цена: $0.04 (Recraft) / $0.025 (FLUX)

**Тип 2 — Цитата (25%)**
- Текст на однотонном фоне
- БЕЗ фото (только текст + декор)
- Цвета: бордовый/беж/чёрный/синий
- Шрифт: Serif для цитаты, sans-serif для подписи
- Размер: 1080×1080

**Тип 3 — Сцена с персонажами (15%)**
- Фотореалистичная сцена
- Мужчина/пара в эмоциональной ситуации
- Минимум текста
- FLUX + LoRA для фотореализма
- Размер: 1080×1080

## ДОРОЖНАЯ КАРТА ДАЛЬШЕ

### Шаг 1: Тест Направления A (Seedream + несколько reference)
1. Взять 3-4 фото Зинаиды из assets/photos
2. Загрузить все на Replicate Files API
3. Сгенерировать с промптом "текст МЕГАМОЗГ, то же лицо что на референсах"
4. Оценить: кириллица + лицо

### Шаг 2: Тест Направления B (FLUX Fill Inpainting)
1. Взять портрет 10_portrait_v3.jpg
2. Наложить текст через скрипт (правильный шрифт Playfair Display)
3. Создать маску текста (белое на чёрном)
4. Отправить в FLUX Fill с промптом "интегрировать текст в изображение"
5. Оценить: выглядит ли текст частью картинки

### Шаг 3: Тест Направления C (Recraft V3 повторно)
1. Другой промпт (с учётом прошлых ошибок)
2. Style: realistic_image
3. Оценить: даёт ли Recraft то что обещают на форумах

### Шаг 4: Выбор направления
Олег смотрит результаты А, В, С → выбирает лучшее → запускаем в производство.

## ПРАВИЛА ИЗ КОММЕНТАРИЕВ ОЛЕГА
1. Не показывать лицо крупным планом — кожа неестественная (09.07)
2. Half body лучше крупного плана (09.07)
3. Шрифт на референсе — serif (Didone / Playfair Display) (09.07)
4. На референсе девушка ПЕРЕД буквами, буквы как рамка (09.07)
5. **Не делать наложение текста через Python/Pillow** — только через дизайнерские инструменты (09.07)
6. Все портреты — через Replicate + LoRA (09.07)
7. Референс: БЕЛЫЙ фон, не тёмный (09.07)
8. Крупный текст сверху и снизу, рамочная композиция (09.07)
9. **Текст должен читаться чётко** — не сливаться с фоном (09.07)
10. Recraft V3 — лучшая для кириллицы, Ideogram V3 Turbo — запасная (из config.yaml)
11. Всё что пробовали — фиксировать: и хорошее, и плохое (09.07)
12. Реестр — главный источник правды, читать при старте (09.07)
13. **Писать "Его страх" по-русски, не "Ego Страх"** (10.07)
14. Генерация через Replicate API напрямую (10.07)
15. Три варианта сразу, параллельно (10.07)
16. Didone/Modern Serif — идеальный шрифт для ЦА (женщины 18-40) (10.07)
