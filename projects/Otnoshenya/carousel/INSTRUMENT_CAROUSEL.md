# ИНСТРУМЕНТ «КАРУСЕЛЬ»

## ПРИ СТАРТЕ — ЗАГРУЗИТЬ ВСЁ:

1. ✅ Навык `copywriting/designer` (стили, цвета, шрифты, FAL)
2. ✅ Навык `copywriting/brand-font-zinaida` (шрифты, обложки)
3. ✅ Файл `STYLE_EGO_STRAKH.md` (референс обложки)
4. ✅ Файл `STYLE_GPT_IMAGE_2.md` (референс кириллицы)
5. ✅ Тему/фазу из phases.db (копирайтинг)
6. ✅ Инструмент (Детектив/Телепат/Тренажер и т.д.)

## ТЕХНИЧЕСКИЙ ПАСПОРТ КАРУСЕЛИ

| Параметр | Значение |
|----------|----------|
| Платформа | Instagram / VK |
| Формат | Квадрат 1080×1080 |
| Кол-во карточек | 4-5 шт |
| Стиль | тёмная палитра, глубокий графит, терракотовые акценты #C05746 |
| Шрифт | Didone Serif (PlayfairDisplay.ttf) |
| Текст | внутри карточек, без ИИ-генерированного текста |
| Лицо Зинаиды | через LoRA на FAL (trigger: zinaida) |

## МОДЕЛИ ДЛЯ ГЕНЕРАЦИИ

| Задача | Модель | Сервис |
|--------|--------|--------|
| Лицо Зинаиды | `fal-ai/flux-lora` + LoRA zinaida | FAL ($0.025) |
| Фон/атмосфера | `fal-ai/gpt-image-2` | FAL ($0.04-0.06) |
| Текст на карточке | СПОСОБ Б (композитинг) + PlayfairDisplay.ttf | Локально |

## АЛГОРИТМ

1. Получить тему (из фазы phases.db) и инструмент
2. Сгенерировать 4-5 фонов через GPT Image 2 (единый стиль, без текста)
3. Сгенерировать Зинаиду через FLUX + LoRA
4. Наложить текст через композитинг (PlayfairDisplay.ttf)
5. Проверить каждую карточку

## ЧЕК-ЛИСТ ПРИЁМКИ

- [ ] Единый визуальный стиль на всех карточках
- [ ] Лицо Зинаиды консистентно
- [ ] Кириллица без ошибок (композитинг)
- [ ] Шрифт Didone (PlayfairDisplay)
- [ ] Тема раскрыта от карточки к карточке
- [ ] Инструмент (CTA) в последней карточке
- [ ] Формат 1080×1080
