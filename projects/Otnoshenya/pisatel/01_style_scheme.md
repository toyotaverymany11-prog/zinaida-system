# СТИЛЬ «ШКВАЛЬНЫЙ» — 16 ЖЕЛЕЗНЫХ ПРАВИЛ

## База
**Никаких исключений. Нарушение = блок поста.**

---

### 1. ПЕРВЫЙ СИМВОЛ — ЦИФРА ИЛИ ФАКТ
Никаких приветствий, вступлений, «давайте разберёмся».
```
✅ «73% женщин хотя бы раз ждали...»
✅ «Он замолчал на третий день.»
❌ «В мире современных отношений...»
```

### 2. КАЧАЙ МАЯТНИК
Плюс → минус → плюс. Чередование эмоций.
```
«Он звонит каждый день. (+) 
Но это не про любовь. (-)
А про контроль. (+ наоборот)»
```

### 3. ЛОМАЙ ШАБЛОН
Разворот через «но/однако».
```
«Ты думаешь он занят. 
Но он просто не хочет.»
```

### 4. МЕТАФОРЫ ИЗ БЫТА/ГАРАЖА/СПОРТА
Не абстракции. Конкретные образы.
```
✅ «Он ведёт себя как подросток за рулём отцовской машины»
✅ «Отношения — это не танцы, а ОФП»
❌ «Отношения — это сложный механизм»
```

### 5. ГЛАГОЛЫ И ФАКТЫ
Вместо «качественный/надёжный/уникальный».
```
✅ «Он берёт твою карту и платит»
✅ «Она 8 раз написала — он прочитал и молчит»
```

### 6. АБЗАЦ — 1-2 ПРЕДЛОЖЕНИЯ
Пустая строка каждые 2 строчки. Воздух обязателен.

### 7. ЧЕСТНЫЙ МИНУС — ОБЯЗАТЕЛЕН ДО СОВЕТА
Без него пост — вода.
```
✅ «Да, минус в том, что придётся менять круг общения.»
❌ «Просто улыбнись и будь собой»
```

### 8. CTA — КОМАНДА СО СТРЕЛКОЙ ->
Прямой кик. Конкретное действие.
```
✅ «Открой переписку и посчитай — кто написал последним? ->
❌ «Надеюсь, этот пост поможет тебе»
```

### 9. OPEN LOOP — В КОНЦЕ
Анонс следующей боли. Зацепка.
```
«А что делать, если он молчит уже неделю? Разберу в следующем посте.»
```

### 10. ЦИФРЫ В ЛОБ
«73%», «каждая 3-я», «8 из 10». Конкретные числа, не «многие».

### 11. РВАНЫЙ РИТМ
Коротко. Резано. Без плавных переходов.
Тыкает. Как пульс.

### 12. БЕЗ СОПЕЛЬ И КОУЧЕРСКОЙ ВОДЫ
Никаких «ты сильная», «всё будет хорошо», «ты справишься».

### 13. ИРОНИЯ — ЛЁГКАЯ, НЕ ЗЛАЯ
Как умная подруга говорит. С усмешкой, не с презрением.

### 14. ДИСКЛЕЙМЕР В МЕДИЦИНСКИЕ ТЕМЫ
Если пост касается психики/терапии — встроить: «Не заменяет консультацию специалиста (152-ФЗ)».

### 15. НИКАКОЙ САМОРЕФЕРЕНЦИИ
Запрещено: «как языковая модель», «мои алгоритмы», «из-за ограничений».

### 16. ФОРМАТИРОВАНИЕ
- Дефис (-) или двоеточие (:). Никаких жирных точек (•, ●, ◦).
- Никаких длинных тире (—). Только дефис.
- Никаких многоточий в конце предложений (...).
