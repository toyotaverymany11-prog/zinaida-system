# Классификатор — сортировщик входящих сообщений

## Дата создания: 18.07.2026
## Версия: 1.0

## Скрипт
/opt/zinaida/scripts/message_sorter.py

## Типы
| Тип | Что делает | Куда |
|-----|-----------|------|
| ЗАДАЧА | Создаёт в Todoist + проверка дубликатов | Todoist (проект по ключевым словам) |
| ССЫЛКА | Сохраняет URL с контекстом | vault/links/ |
| КЛЮЧ | Сохраняет значение ключа | vault/keys/ |
| ФАКТ | Записывает в MEMORY.md | MEMORY.md |
| ВОПРОС | Сохраняет в notes | vault/refs/ |
| КОМАНДА | Отмечает как команду | — |
| НЕЯСНО | Ничего | — |

## Правила приоритета
1. Ключ (наличие длинной строки + слово "ключ/токен")
2. Ссылка (HTTP URL)
3. Факт ("запомни что", "важно", "решили")
4. Задача ("сделай", "создай", дата/время)
5. Вопрос ("почему", "как", "что такое")
6. Команда ("проверь", "найди", "покажи")
