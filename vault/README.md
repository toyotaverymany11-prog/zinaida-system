# Vault — гарантированное хранение ссылок, ключей, паролей

## Структура
- `/opt/zinaida/vault/links/` — все ссылки с контекстом
- `/opt/zinaida/vault/keys/` — API ключи и пароли
- `/opt/zinaida/vault/configs/` — бэкапы конфигов

## Железное правило (16.07.2026)
При получении ЛЮБОЙ ссылки/ключа/пароля от Олега:
1. Сохранить в `/opt/zinaida/vault/links/` или `/opt/zinaida/vault/keys/` с датой и контекстом
2. Записать в Mem0
3. Записать в updates_log.md
4. Если ключ — проверить что он уже есть в .env файлах
