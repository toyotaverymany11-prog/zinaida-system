## 1. HERMES + FAL: КАК РАБОТАЕТ ИНТЕГРАЦИЯ

### Официальная документация Hermes
Hermes Agent из коробки поддерживает **FAL.ai** как основной провайдер генерации изображений через встроенный `image_generate` tool. 11 моделей доступны сразу.

**Два способа настройки:**
1. **FAL API Key (наш вариант)** — `FAL_KEY` в .env + `hermes tools` для выбора модели
2. **Nous Portal Subscription** — платная подписка, не нужен FAL ключ

**Конфиг (уже стоит у нас):**
```yaml
image_gen:
  provider: fal                          # ✅ мы переключили
  model: fal-ai/flux-2/klein/9b         # ✅ дефолтная, $0.006
  use_gateway: false                     # false для прямого FAL
```

**Как менять модель:** `hermes tools` → Image Generation → выбрать из таблицы

Запущен **21 апреля 2026**. OpenAI называет его «generalist model for images».

**Характеристики кириллицы:**

**Скорость:** ~20 секунд
**Плюсы:** SOTA текст, фотореализм, редактирование

## 3. ТИПИЧНЫЕ ОШИБКИ ПОЛЬЗОВАТЕЛЕЙ FAL (ИЗ TRUSTPILOT + REDDIT)

### Trustpilot: ⭐ 2.4/5 (18 отзывов, 72% — 1 звезда)

**⚠️ Главная проблема: BILLING SHOCK**
- **$110 неожиданный счёт после $10 кредита** — пользователь положил $10, но FAL начислял по postpaid (постоплата). Кредит сгорел, а счёт пришёл сверху.
- **Как защититься:** НЕ использовать postpaid. Только prepaid (предоплата). Смотреть billing dashboard.

**⚠️ Ключевые ошибки (проверено на нас):**
1. **API key не имеет доступа к модели** — разный scope (API vs ADMIN). На некоторых моделях (mmaudio) нужен доступ.
2. **Токен протух** — в Replicate тоже было. FAL вроде стабильнее.
3. **Credit expiry** — баланс может сгореть без предупреждения (как у Zac Garton — $2,000 → 0 без уведомления).
4. **Model not found** — если модель удалена или переименована.
5. **FAL_KEY не установлен** — самая частая ошибка новичков.

**⚠️ Рекомендации пользователей:**
- «The tech is worth exploring. The support is not.» — поддержка не отвечает
- Ставьте spending limits (лимиты трат)
- Проверяйте счёт после каждой сессии
- Не храните API key в коде — только в .env (мы так и делаем)
- Для русского текста: **GPT Image 2** — единственная с надёжной кириллицей
