# 8003 (Zina2-Router) — умный классификатор Flash/Pro

## Что это

8003 — НЕ «просто DeepSeek Chat». Это **интеллектуальный роутер**, который сам решает:
- **Flash (deepseek-chat)** — для односложных ответов (привет/да/нет/ок/спасибо)
- **Pro (deepseek-reasoner)** — для ВСЕГО осмысленного

## Как работает классификация

Файл: `/opt/zinaida/meta_agent/zina2_router.py`

### Flash (быстрый, дешёвый)
Только если сообщение:
- <= 2 слов И все слова из FLASH_ONLY_WORDS
- FLASH_ONLY_WORDS = {"привет", "пока", "да", "нет", "ок", "окей", "ладно", "ага", "спасибо", "понял", "поняла", "хорошо", "норм", "здравствуй", "прив"}

### Pro (глубокий, думает) — всё остальное
ЛЮБОЙ из триггеров → Pro:
- Вопросы: что, как, почему, зачем, где, когда, какой, сколько
- Анализ: анализ, разбери, объясни, расскажи, сравни, проверь, исправь
- Код: код, скрипт, python, api, json, sql, баг, дебаг
- Контент: напиши, пост, контент, текст, история, креатив, слоган
- Стратегия: стратегия, план, решение, сложный, трудный
- **Длина > 150 символов** → автоматически Pro

Чувствительность Pro — МАКСИМАЛЬНАЯ. Малейший намёк на сложность → Pro.

## Как подключён в Telegram bot

В bot.py (v2, 12.07.2026) каждый запрос идёт с:
```
provider: custom:zina2-router
model: Zina2-Router
```

Это значит Hermes Gateway пересылает запрос на порт 8003, где классификатор решает Flash или Pro.

## Как подключён в Hermes Studio (default profile)

Hermes Studio профиль `default` показывает `model: deepseek-chat`. Это НЕ 8003 — это прямой DeepSeek Flash.

**Разница:**
- Telegram bot: `custom:zina2-router` → 8003 → умный классификатор (Flash для мелочи, Pro для всего остального)
- Hermes Studio default: `deepseek-chat` → прямой DeepSeek Flash (всегда Flash)

Если в Hermes Studio нужно использовать 8003 — указывать `provider: custom:zina2-router, model: Zina2-Router`.
