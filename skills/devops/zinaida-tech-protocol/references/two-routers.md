### Архитектура: классификация → генерация → усиление

```
Запрос
  │
  ▼
┌──────────────────────────────┐
│ 1. КЛАССИФИКАТОР             │
│    приветствия → Ollama       │
│    обычные     → DeepSeek Fl. │
│    сложные     → DeepSeek Pro │
└──────────┬───────────────────┘
           │
           ▼
┌──────────────────────────────┐
│ 2. ПРЕ-ПРОЦЕССИНГ (Mistral)  │  ← бесплатно
│    чистка запроса, структура  │
└──────────┬───────────────────┘
           │
           ▼
┌──────────────────────────────┐
│ 3. ГЕНЕРАЦИЯ (DeepSeek)      │  ← платно
│    + параллельный RAG-поиск   │  ← бесплатно, одновременно
└──────────┬───────────────────┘
           │
           ▼
┌──────────────────────────────┐
│ 4. ВЕРИФИКАЦИЯ (Mistral)     │  ← бесплатно
│    галлюцинации, логика       │
└──────────┬───────────────────┘
           │
           ▼
┌──────────────────────────────┐
│ 5. РЕДАКТУРА (GigaChat)      │  ← бесплатно
│    русский, AI-штампы         │
└──────────┬───────────────────┘
           │
           ▼
┌──────────────────────────────┐
│ 6. FALLBACK                  │
│    DeepSeek упал → Mistral    │
│    Mistral упал → Ollama      │
└──────────────────────────────┘
```

### Ключевой принцип: Бесплатные = УСИЛИТЕЛИ, не генераторы

- **DeepSeek Flash/Pro** — единственный генератор мощных ответов
- **Mistral** — не генерирует ответы (кроме fallback), а:
  - чистит запрос перед отправкой (пре-процессинг)
  - проверяет ответ на галлюцинации (верификация)
  - ищет факты в RAG параллельно (поиск)
- **GigaChat** — правит русский язык, убирает AI-маркеры
- **Ollama** — отвечает только на приветствия «привет/пока/да/нет»

### Чувствительность Pro (v2.0 — снижена)

| Было (v1.0, 8003) | Стало (v2.0, 8005) |
|-------------------|-------------------|
| Pro при 150 символах | Pro при 300 символах |
| Pro при любом вопросе | Pro только по триггер-словам (анализ, код, сравнение) |
| Почти всё уходило в Pro | Flash = дефолт для 80% запросов |

### Классификация запросов (v2.0)

| Тип | Класс | Модель | Стоимость |
|-----|-------|--------|-----------|

| >300 символов | pro | DeepSeek Pro | $1.42/M |
| Код (символы {};:= > 40% латиницы) | pro | DeepSeek Pro | $1.42/M |
