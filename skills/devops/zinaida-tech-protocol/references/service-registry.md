# Реестр сервисов — опыт работы (шпаргалка для навыков)

Файл: `/opt/zinaida/shared_memory/service_registry.md`
Правило проверки: AGENTS.md, строки 7-8.

## Как использовать
Перед предложением ЛЮБОГО внешнего сервиса — прочитать `/opt/zinaida/shared_memory/service_registry.md`.
Если сервис уже ❌ не работает — НЕ ПРЕДЛАГАТЬ. Сразу называть альтернативу из ✅.
После пробы нового сервиса — сразу дописывать результат в реестр.
