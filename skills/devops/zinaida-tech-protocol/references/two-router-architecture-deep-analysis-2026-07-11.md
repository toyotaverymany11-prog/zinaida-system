## ВЫВОДЫ ПО КАЖДОМУ КОМПОНЕНТУ

### Zina2-Router (8003) — DeepSeek-only ✅
**Плюсы:**
- Чистый, компактный код (363 строки), легко читать
- Хорошая логика выбора Flash/Pro (PRO_CHAR_THRESHOLD + триггер-слова)
- Graceful fallback при ошибках
- DeepSeek V4 Flash реально работает, отвечает за 3-5 сек

**Минусы:**
- Только DeepSeek (платный) — нет интеграции с бесплатными провайдерами
- Нет QA-пайплайна (верификация/редактура)
- Нет кэширования
- Нет ротации ключей
- «Reasoning баг» на короткие запросы (1-2 слова)

### Zinaida-Router (8002) — мульти-провайдер ❌ перегружен

- GigaChat работает через него

## ПРЕДЛОЖЕНИЕ: НОВЫЙ ЕДИНЫЙ РОУТЕР v2

### Концепция «Бесплатный ансамбль»
Объединить лучшее из всех трёх компонентов в один роутер (~1000 строк):

```
                    ┌──────────────────┐
                    │     ЗАПРОС       │
                    └────────┬─────────┘
                             │
                    ┌────────▼─────────┐
                    │  КЛАССИФИКАТОР   │
                    │  (тип задачи)     │
                    └───┬──────┬───────┘
                        │      │
            Короткий  ──┤      ├──  Сложный/анализ
         (<50 символов)  │      │  (>300 символов)
                        ▼      ▼
              ┌──────────┐ ┌──────────┐
              │ Ollama   │ │ DeepSeek │
              │ gemma3:4b│ │ Flash    │
              │ (0 руб)  │ │ ($0.27)  │
              └─────┬────┘ └─────┬────┘
                    │            │
                    ▼            ▼
              ┌──────────────────────────┐
              │   QA-ПРОВЕРКА (встроена) │
              │  1. Mistral — верификация│
              │  2. GigaChat — редактура │
              └──────────┬───────────────┘
                         │
                    ┌────▼────┐
                    │  ОТВЕТ  │
                    └─────────┘
```

### Ключевые фичи нового роутера:
1. **Автоматическая ротация ключей** — Mistral (5 ключей, при 429 → следующий)
2. **Встроенный QA-пайплайн** — Mistral верифицирует → GigaChat правит русский
3. **Кэширование** — хеш messages → response, TTL 5 мин
4. **Graceful fallback** — если платный не отвечает → бесплатный
5. **Логирование** — все запросы в analytics.db
6. **Единый endpoint** — `/v1/chat/completions`

### Логика выбора модели:
| Тип запроса | Модель | Стоимость | Приоритет |
|------------|--------|-----------|-----------|
| Приветствие, «да/нет» | Ollama gemma3:4b | 0 | высший |
| Простой вопрос | Mistral-small | 0 | высокий |
| Средний (русский) | Mistral-large / GigaChat | 0 | средний |
| Сложный/анализ | DeepSeek Flash (V4) | $0.27/M | низкий |
| Глубокий разбор | DeepSeek Pro (reasoner) | $1.42/M | самый низкий |
| Гонка (race) | Mistral vs DeepSeek Flash | min | для ответственных |
