## Детали по каждому

### Mistral
- **API:** api.mistral.ai/v1
- **Free Tier:** ~1 млрд токенов/месяц (по факту безлимит для наших объёмов)
- **Rate limit:** 1 запрос/сек на ключ. 3 ключа = 3 req/сек
- **Дата сброса:** ежемесячно с даты регистрации аккаунта. Точную дату не узнали — не заходили в admin.mistral.ai/plateforme/limits (нужен логин/пароль)
- **Сигнал проблемы:** HTTP 429 в логах роутера

### GigaChat
- **API:** gigachat.devices.sberbank.ru/api/v1 (chat), ngw.devices.sberbank.ru:9443/api/v2/oauth (auth)
- **Freemium:** 1 млн токенов/месяц (1M = ~3.5 млн символов)
- **Дата сброса:** каждый месяц с даты регистрации проекта на develop.sber.ru
- **Почему 402:** лимит исчерпан. Даже `GigaChat:latest` (бесплатная текстовая) даёт 402
- **Проверка обновления:** запускать тестовый запрос раз в неделю. Как перестанет быть 402 — обновился. Записать дату.
- **Важно:** `/tokens/balance` API доступен только при платном пакете (403 на Freemium)

### DeepSeek
- **Баланс:** $17.45
- **Расход:** $0.27/1M токенов Flash, $1.42/1M Pro
- **Обновление:** пополнение вручную
