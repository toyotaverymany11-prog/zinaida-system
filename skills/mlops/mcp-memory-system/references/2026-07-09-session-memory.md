# Сессия 09.07.2026 — создание MCP Memory Server

## Контекст
Олег хотел улучшить мою память. Предложил Obsidian — у него приложение на iPad, но он не захотел в нём разбираться.

## Альтернативы (почему не Obsidian)
1. Obsidian на сервер — не нужен, Obsidian это GUI для людей
2. Obsidian vault на сервере (папка .md) — создали, но Олег не хочет трогать iPad
3. Obsidian Local REST API — плагин на iPad, но Олег не хочет ничего устанавливать/настраивать

## Почему выбрали MCP
1. Не зависит от версии Hermes — отдельный процесс
2. MCP протокол — Hermes поддерживает нативно через config.yaml
3. SQLite WAL — надёжно, бекапится копированием
4. FTS5 — быстрый поиск по русскому тексту
5. Граф через таблицу links — ассоциативная память
6. Безлимитный объём

## Результат
Сервер написан и работает. MCP инструменты зарегистрированы. B Конфиг Hermes добавлен.
