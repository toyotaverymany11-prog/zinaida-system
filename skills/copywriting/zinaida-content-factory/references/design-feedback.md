# Дизайн-фидбек: запись оценок Олега

Файл: `/opt/zinaida/shared_memory/design_feedback.md`

**Когда писать:** после каждой оценки Олегом визуала/фото/обложки.
**Что писать:** дата, что оценивал, оценка, почему, пожелания.
**Зачем:** чтобы при следующей генерации учитывать его предпочтения.

Формат записи:
```
## YYYY-MM-DD
### Название файла
- Оценка: 👍/👎
- Почему: причина
- Пожелание: что поменять
```
