1. **Нумерация:** `00_` = главный README, далее по темам
2. **При старте:** читается вся папка целиком
3. **Чёрный список — 18 категорий** (структурные шаблоны, канцелярит, токсичная эмпатия, искусственный клей, визуальный шум, ложная авторитетность, поп-психология, пунктуация, мета-вступления, псевдо-интерактив, хеджирование, успокоительные, SEO-вода, аналогии, риторические вопросы, шаблонные выводы, паразитные усилители, самореференция)
4. **Обработка docx:** извлечь → прочитать → сравнить → новое распределить, дубли пропустить
