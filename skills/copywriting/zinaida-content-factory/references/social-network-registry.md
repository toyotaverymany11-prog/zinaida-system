## Команды быстрого доступа

Оператор говорит `#vkontakte` (или любую другую сеть) — агент читает соответствующую карточку из `inbox/Контент/_social/` и выдаёт:
- Ссылки на страницы/группы
- Текущий статус (открыта/наполняется/запущена)
- Количество подписчиков и постов
- Состояние дизайна
- Последние действия
- Что нужно сделать следующим

Формат команды: `#<имя_сети>` в нижнем регистре, слитно.

## Стандартный шаблон карточки сети

```markdown
# НАЗВАНИЕ СЕТИ
## Карточка сети

### Тип
Группа / страница / канал / бот

### Ссылки
- **URL:** ссылка
- **ID:** идентификатор

### Статус
✅/⬜ Открыта / Наполняется / Запущена / Дизайн / Аналитика

### Описание и наполнение
Текущее состояние, что уже сделано, что опубликовано

### Дизайн
Есть/нет, что сделано

### API-доступ
Настроен/не настроен, какой токен нужен

### Что нужно сделать
- [ ] Пункты задач

### Заметки
Форматы, особенности аудитории, API

Обновлено: ДАТА
```

## Архитектурная рекомендация (из сессии)

НЕ создавать отдельного агента по соцсетям. Основной агент (Зинаида) управляет всеми сетями:
- Через браузер — просмотр страниц, настроек, аналитики
- Редактирование — напрямую в веб-интерфейсе соцсети
- Делегация agent2 (кодеру) — только автоматизация (скрипты публикации, парсинг аналитики)
- Делегация Лере (дизайнеру) — обложки, аватары, визуал
