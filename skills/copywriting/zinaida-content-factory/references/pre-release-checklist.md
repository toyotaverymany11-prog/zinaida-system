## ЗАПРЕЩЁННЫЕ ФРАЗЫ (НАВСЕГДА)

| Фраза | Почему | 
|-------|--------|
| «разберу по косточкам» | Блевотный штамп |
| «ни один AI не делает такого» | Самовосхваление |
| «это не гадание / не таро» | Паттерн "это не Х, это Y" |
| «пришлю ссылку» | Заменять на "предоставлю инструмент" |
| Две CTA в одной фразе | Размывает действие |
