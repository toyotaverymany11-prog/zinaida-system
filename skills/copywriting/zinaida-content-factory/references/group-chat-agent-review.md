# Group Chat Multi-Agent Review — Hermes Studio API

Как подключить внешнего агента (Abacus / любой профиль) к обсуждению текста для улучшения стиля, юмора, речевых оборотов.

Дата: 2026-07-11

## API Эндпоинты

Hermes Studio v0.6.28 поддерживает групповые чаты с участием агентов:

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/hermes/group-chat/rooms` | Создать комнату (требуется: name, inviteCode) |
| GET | `/api/hermes/group-chat/rooms` | Список комнат |
| POST | `/api/hermes/group-chat/rooms/{roomId}/agents` | Добавить агента (требуется profile) |
| GET | `/api/hermes/group-chat/rooms/{roomId}/agents` | Список агентов в комнате |
| DELETE | `/api/hermes/group-chat/rooms/{roomId}/agents/{agentId}` | Удалить агента |

## Как это использовать для ревью текста

1. **Создать групповую комнату:**
   `POST /api/hermes/group-chat/rooms` с body: `{"name": "Ревью поста", "inviteCode": "post-review-1"}`

2. **Добавить агента-критика:**
   `POST /api/hermes/group-chat/rooms/{roomId}/agents` с body: `{"profile": "abacus"}` (или имя другого профиля)
   Агент = любой зарегистрированный профиль Hermes (agent2, lera, или внешний).

3. **Зинаида + Агент обсуждают пост:** каждый участник комнаты видит сообщения друг друга. Агент-критик даёт рекомендации по формулировкам.

## Требования к агенту

- Агент должен быть зарегистрирован как профиль Hermes (в `/root/.hermes/profiles/<name>/`)
- Агент должен иметь свой SOUL.md и навыки
- Если агент не зарегистрирован — его нужно сначала создать через стандартную процедуру создания профиля

## Альтернативный подход (без Group Chat)

Если Group Chat не подходит — можно использовать `delegate_task`:
```python
delegate_task(
    goal="Дай рекомендации по улучшению этого поста: [текст поста]. Нужно: 1) Улучшить юмор (убрать AI-шный, добавить тонкий, неожиданный), 2) Усилить речевые обороты, 3) Добавить фразу для пересылки подруге",
    context="Ты креативный копирайтер-комик. Целевая аудитория: женщины 25-45, РФ, психология отношений.",
    toolsets=["terminal", "file"]
)
```
Суб-агент вернёт рекомендации, которые можно применить к посту.
