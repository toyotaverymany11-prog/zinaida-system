# Post Analysis Checklist (Обушение-анализ)

Ручной чек-лист для анализа поста на все 7 слоёв сборки. Замена несуществующему `post_analyzer.py`.

## SQLite helper — получить последние посты для анализа

```bash
# Из content_rotation.db
sqlite3 /opt/zinaida/inbox/PROJECTS/Otnoshenya/content_rotation.db \
  "SELECT id, platform, substr(content, 1, 200) AS snippet, created_at FROM posts ORDER BY created_at DESC LIMIT 5;"
```

## Layer-by-Layer

### Слой 0 — DNA
- [ ] Голос Зинаиды (28 лет, Сочи, без розовых очков)?
- [ ] Цифры и данные, а не гадание?
- [ ] Без коучерской воды?
- [ ] Глаголы в женском роде?

### Слой 1 — Style
- [ ] Первый символ — цифра или факт?
- [ ] Маятник: плюс → минус → плюс?
- [ ] Есть "но/однако" (разворот)?
- [ ] Метафоры из быта/гаража/спорта?
- [ ] Абзацы 1-2 предложения, пустая строка каждые 2 строки?
- [ ] Честный минус перед советом?
- [ ] Рваный ритм?
- [ ] Open Loop в конце?

### Слой 2 — Phase
- [ ] Фаза определена из phases.db?
- [ ] Соответствует теме поста?

### Слой 3 — Attack
- [ ] Хук (боль целевой аудитории)?
- [ ] Статистика/цифра?
- [ ] Микро-история?
- [ ] Все три сварены в один абзац?

### Слой 4 — RAG
- [ ] Запрос делался в smm_rag.db?
- [ ] Чанк релевантен теме?
- [ ] Данные не устарели?

### Слой 5 — CTA
- [ ] Платформа определена? TG/MM → инструмент, ост. → сохранить/переслать?
- [ ] Если TG/MM: инструмент выбран по фазе/боли поста? (см. `references/tool-cta-strategy.md`)
- [ ] Словарь боли целевой аудитории?
- [ ] CTA — прямая команда со стрелкой (→)?
- [ ] Нет коучерского «попробуй, тебе понравится»?
- [ ] Формулировка в стиле Зинаиды (вызов, а не предложение)?

### Слой 6 — JSON
- [ ] Формат правильный (если сохранён как JSON)?
- [ ] Поля: content, platform, scheduled_date?

## Blacklist Scan
- [ ] Нет • ● — (длинное тире) … — в тексте?
- [ ] Нет "давайте разберёмся", "важно отметить", "стоит понимать"?
- [ ] Нет "это не просто X а Y", "больше чем просто"?
- [ ] Нет "надеюсь поможет", "подводя итог"?
- [ ] Нет мужского рода глаголов?
- [ ] Нет "как языковая модель", "оптимизировать процессы"?
