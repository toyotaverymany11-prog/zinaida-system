# Error Catalog — 22+ Lessons Learned (2026-07-11)

## Critical Writing Errors

1. **Выдумала цифру 72%** — не проверила источник. **Фикс:** каждая цифра → верификация через statistics_base или веб.
2. **Формат «читательница Анна» на пустом канале.** **Фикс:** форматы с «реальными людьми» только с 3-4 недели.
3. **Пост не на тему** — «приборная панель» вместо «он молчит». **Фикс:** перед написанием проверить боль ЦА.
4. **Не указала фазу** — нет [ФАЗА: X]. **Фикс:** маркер фазы обязателен.
5. **Не сделала RAG-запрос.** **Фикс:** перед написанием rag_query по теме.
6. **Сценарии по 5 строк** — теряется ритм. **Фикс:** прочитать вслух после написания.
7. **Длинные тире U+2014 вместо дефиса.** **Фикс:** верификатор обновлён.
8. **Контент-план без привязки к инструментам.** **Фикс:** апдейтнуть план.
9. **Посты однотипные на 8 платформ.** **Фикс:** каждой платформе — свой угол.
10. **Нет верификатора в процессе.** **Фикс:** VERIFIER_CHECKLIST.md обязателен.
11. **Слово «имитируют» слишком агрессивное.** **Фикс:** мягче начало.
12. **«=» посреди текста.** **Фикс:** убрать.
13. **Молодёжный сленг (игноришь).** **Фикс:** стилистика под платформу.
14. **Нет иронии — сухой текст.** **Фикс:** проверять характер.
15. **Не проверила ссылку перед отправкой.** **Фикс:** curl-проверка.
16. **Нет RAG-запроса (повтор).** **Фикс:** FTS5 запрос перед написанием.
17. **Утверждала что OpenRouter мёртв — не перепроверила.** Oleg дал новые ключи, инструкцию. Старая информация (IP заблокирован) подтвердилась, но я должна была перепроверить СВОИМИ ГЛАЗАМИ перед ответом. **Фикс:** все утверждения о статусе провайдеров/ключей перепроверять live-запросами перед ответом.
18. **Пост «бледный», без юмора.** Oleg: «хуета какая-то бледная, не цепляет, юмора нет». **Фикс:** добавить 1-2 неожиданных оборота, живые метафоры, разрушение шаблонов.
19. **Длинные тире (—) в тексте (повтор).** **Фикс:** заменить на дефис (-) ВЕЗДЕ. Проверять через python3 с ord(c)==8212.
20. **Хештегов 5 вместо 2-3.** **Фикс:** макс 3 для VK 2026.
21. **Разделители —— в тексте.** **Фикс:** убрать, оставить пустые строки.
22. **Стрелка -> в тексте.** **Фикс:** заменить на обычное тире или убрать — может быть сигналом «реклама» для ML.
