# ИССЛЕДОВАНИЕ: СИСТЕМЫ КАЧЕСТВА УСПЕШНЫХ КОПИРАЙТЕРОВ

**Источники:**
1. Texterra — система контроля качества ИИ-контента (2025)
2. PPC World — чек-лист проверки текста на нейросеть (7 сервисов)
3. Tonemark.ai — психология вирусных хуков, STEPPS framework (Jonah Berger)
4. Stephen King — "On Writing", "kill your darlings"
5. SCAMPER — система креативных трансформаций

## Почему AI-тексты выглядят как AI

| Паттерн | Описание |
|---------|----------|
| Слишком ровный стиль | Без ошибок, без пауз, без нестандартных выражений |
| Поверхностная логика | ИИ пишет красиво, но без глубины |
| Шаблонные конструкции | "в современном мире", "одним из важнейших аспектов" |
| Синтаксическая гладкость | Нет рваного ритма, нет коротких ударов |
| Однообразный словарь | Либо слишком бедный, либо неестественно редкие слова |
| Отсутствие личного опыта | Текст обезличен и сух |

**Вывод:** AI-текст выдаёт не отсутствие "воды", а отсутствие человеческой неровности.

## Фреймворк вирусных постов (STEPPS)

| Фактор | Суть | В переводе на наши посты |
|--------|------|------------------------|
| Social Currency | Делиться = выглядеть умной | "Скинь подруге - она поймёт" |
| Triggers | Привязать к повседневности | "Понедельник. Он не пишет. Опять." |
| Emotion | Высокое возбуждение | Не "интересно", а бесит/обидно/смешно |
| Practical Value | Применимость | "Бери и делай" |
| Stories | Нарратив | История, а не лекция |

## Трёхуровневая система контроля

| Уровень | Кто | Что делает |
|---------|-----|------------|
| Writer | Зинаида | Пишет черновик: хук + история + инструмент |
| Editor | Агент-анализатор | Проверяет AI-паттерны, факты, стиль |
| Chief | Олег | Финальное утверждение |

## Kill Your Darlings

Любимые фразы, которые надо вырезать:
- "Я разработала специальный инструмент..." → можно короче
- "Это не гадание..." → AI-паттерн, вырезать навсегда
- "Ни один AI не делает такого..." → вырезать
- "Напиши Х в комментариях..." + "или в личку" → одну CTA, не две

## Двухэтапная проверка перед публикацией

**Этап 1 — во время написания:**
1. Первая строка — останавливает скролл?
2. Вторая строка — раскрывает суть?
3. Первая строка цепляет максимум людей, не сужает?
4. Есть ли "Haha" (смех) или "Wow" (удивление)?
5. Есть ли фраза, которую хочется переслать?

**Этап 2 — перед публикацией (анализатор):**
1. AI-паттерны (16 проверок)
2. Запрещённые фразы (из истории правок)
3. Длина под платформу
4. Связность (нет разнобоя)
5. Инструмент назван капсом, показана ценность
6. Одна CTA
