# Анимация GIF для триггеров — инструкция (13.07.2026)

## Файлы
- `/opt/zinaida/design/system_activate.gif` — основная анимация активации
- `/opt/zinaida/design/make_activate_gif.py` — скрипт генерации

## Размер и формат
- ЖЁСТКО: **1920×120** (узкая полоса во всю ширину экрана)
- 45 кадров, 15 FPS, 3 секунды
- НЕ квадрат (800×500) — Олег ругается

## Если Олег говорит «маленький квадратик / не видно / плохой размер»
1. НЕ обсуждать
2. НЕ спрашивать «какой размер?»
3. НЕ пытаться отредактировать GIF руками
4. Просто запустить: `python3 /opt/zinaida/design/make_activate_gif.py`
5. Проверить: `file /opt/zinaida/design/system_activate.gif` — должно быть `1920 x 120`

## Структура GIF (make_activate_gif.py)
- Бегущая сканирующая линия слева направо (циан #00CCFF)
- Эхо-линия с отставанием на 20%
- Матричные шумовые полоски
- Пульсирующие края сверху и снизу
- Текст «СИСТЕМА АКТИВИРОВАНА» с мерцанием (появляется на 30%)
- Подтекст «▸ ЗАПУСК [КОНТЕКСТ]-МОДУЛЯ ◂» (розовый)
