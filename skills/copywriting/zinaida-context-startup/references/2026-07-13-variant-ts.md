# Вариант Ц (гибридная воронка) — утверждено 13.07.2026

## Суть
**Канал (контент, прогрев) + бот (продукт, инструменты).**
Канал создаёт спрос — бот закрывает потребность.

## Схема
```
Слой 1: Канал/соцсети — контент, прогрев, доверие
  ↓
Пост про 1 из 6 инструментов
  ↓
CTA: «Напиши [СЛОВО] — дам доступ к инструменту бесплатно»
  ↓
Пользовательница пишет в личку/комментарии
  ↓
Зинаида (AI-бот) отвечает, переводит в Telegram-бота
  ↓
Слой 2: Telegram-бот / MessengerMax — продукт
  ↓
Инструменты + память + токены + подписка
```

## Ключевые принципы
- Везде Зинаида — один персонаж, пользователь не видит разрыва
- Нативная продажа: показать ценность инструментов, не рекламировать подписку
- Автоворонка 1 дня (не 7): бесплатный доступ → демонстрация → paywall
- Telegram = основа, MessengerMax = дубль
