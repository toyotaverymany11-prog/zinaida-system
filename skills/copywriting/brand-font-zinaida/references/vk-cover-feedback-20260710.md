## Ключевые правила (запомнить!)

1. **Serif шрифт на ночном skyline = НИКОГДА.** Олег ненавидит.
2. **При запросе «докрути» — найди существующие файлы, не начинай новый тип генерации.**
3. **Nano Banana 2 — единственная рабочая модель для кириллицы** на VK обложках.
   Всегда даёт квадрат 1024×1024. Для wide-формата → crop центра.
4. **FLUX Dev не пишет кириллицу** без LoRA.
5. **Recraft V3 пишет латиницу** если промпт на английском.
