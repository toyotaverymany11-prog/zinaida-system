## Replicate Face LoRA — попытка

- **Проблема:** Flux-1.1-pro не сохраняет лицо (rendom face каждый раз)
- **Исследование:** https://replicate.com/blog/generate-consistent-characters — 4 модели для face consistency
  - Все дают 403 с нашим токеном (error code 1010)
- **Решение:** LoRA fine-tuning через `replicate/fast-flux-trainer` (~$2, ~2 мин)
- **Затык:** Токен не даёт создавать модели через API (403). Нужно создать модель через веб
- **Баланс счёта:** ~$10
- **Текущий статус:** Нужно создать модель `zinaida-face-lora` на replicate.com/create (Private, GPU T4)
- **Ошибка:** Сгенерировала фото с брюнеткой, потому что паспорт описывал вымышленную внешность. Реальная Зинаида — **платиновый блонд, голубые глаза, спортивная фигура**. Урок: сверять генерацию с оригиналами через vision, не доверять тексту в паспорте.

## Правила, добавленные за сессию

1. **Партнёр, не исполнитель** — всегда высказывать мнение и рекомендации
2. **Без технических размышлений в чате** — никаких «Let me check...», «Good, now I need...» — делать молча, писать итог
3. **Не урезать ответы в Telegram** — канал доставки контента, не трогать без согласования
4. **Проверять генерацию по оригиналам** — не доверять текстовому описанию больше чем реальным фото
