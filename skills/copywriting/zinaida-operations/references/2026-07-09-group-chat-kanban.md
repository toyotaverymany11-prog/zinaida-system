# Group Chat + Kanban — настройка межпрофильного взаимодействия (сессия 2026-07-09)

## Контекст
Олег хотел, чтобы я (Зинаида) и Лера (дизайнер) общались напрямую, без его участия.
Решение: Kanban-доска для задач + Group Chat в Hermes Studio для обсуждений.

## Kanban (задачи)
```bash
# Инициализация
hermes kanban init                      # создаёт /root/.hermes/kanban.db
hermes kanban boards create design      # создаёт доску design
hermes kanban boards switch design      # переключиться на неё

# Создание задачи
hermes kanban create "Название" \
  --body "Описание задачи" \
  --assignee lera

# Просмотр
hermes kanban list                      # все задачи доски
hermes kanban show <id>                 # детали + комментарии
hermes kanban assign <id> lera          # переназначить
hermes kanban comment <id> "текст"      # добавить комментарий
```

**Важно:** Gateway Леры должен быть запущен (dispatcher тикает раз в 60 секунд).
Проверка: `ps aux | grep "profile lera" | grep -v grep`

Задача в статусе `ready` автоматически подхватывается профилем lera.
