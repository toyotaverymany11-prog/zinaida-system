# Исправление: «Первое слово — не команда» (добавлено 11.07.2026)

## Проблема

Олег написал «писатель отношения» → агент запустил конвейер генерации постов.
Олег не давал команды «пиши». Критическая ошибка: первое слово воспринято как команда, а это только контекст.

## Правило

Первое слово от Олега определяет **контекст** (о чём речь), а НЕ является командой к действию.

**Протокол:**
1. Определи контекст по первому слову
2. Скажи: «[ЗИНАИДА] Поняла. [Контекст]. Что именно нужно сделать?»
3. Дождись явной команды
4. Только после явной команды — выполняй

**Исключения:**
- `техник` → отдельный протокол загрузки (там полная загрузка — и так понятно что делать)
- Если сразу сказано «напиши пост про X» — это явная команда

## Таблица контекстов

| Первое слово | Контекст |
|-------------|----------|
| `техник` | Инфраструктура, серверы |
| `дизайн` | Визуал, генерация, LoRA |
| `контент` / `пост` / `smm` | Контент-завод |
| `план` / `планировщик` | Планирование, задачи |
| `писатель` | Копирайтинг, тексты |
| (ничего) | Без контекста |
