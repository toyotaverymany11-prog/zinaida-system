# Multi-Agent Checker Pattern (11.07.2026)

## Концепция
Бесплатные модели НЕ заменяют платную, а ДОПОЛНЯЮТ её, создавая дополнительные слои качества, которых иначе не было бы.

## Схема (для Default-режима)
1. **DeepSeek V4 Flash** (роутер 8003, платный) — пишет/генерирует основной контент
2. **Mistral** (бесплатно, 3 ключа) — проверяет ответ на галлюцинации, логические дыры, нестыковки
3. **Ollama gemma3:4b** (бесплатно) — ищет в smm_rag.db дополнительные факты, статистику, кейсы
4. **GigaChat** (бесплатно, OAuth2) — проверяет русский язык на естественность, культурные отсылки

## Почему это УСИЛИТ качество
- Модель не видит свои ошибки — свежий взгляд другой модели ловит 90% галлюцинаций
- DeepSeek может «забыть» про RAG — Ollama найдёт и подставит
- GigaChat видит неестественные обороты, которые англоязычная DeepSeek не чувствует
- По данным исследований: multi-agent системы дают +60% точности на сложных задачах

## Когда использовать
- **Default (по умолчанию):** включены все проверки. Время не критично, качество выше
- **Очень высокая:** DeepSeek V4 Pro — полный максимум, без проверок (скорость)

## Бесплатные провайдеры — актуальные статусы
Смотреть навык `devops/provider-audit-reference`. На 11.07.2026:
- ✅ Mistral (3 ключа), Ollama (3 ключа), GitHub Models, GigaChat
- ❌ OpenRouter (403), DeepSeek direct (401), Nous Portal (истёк)

## Предостережение
Не заменять DeepSeek полностью бесплатными моделями для креативных задач (написание постов, CTA, стиль Зинаиды). Бесплатные — только для проверки и дополнения.
