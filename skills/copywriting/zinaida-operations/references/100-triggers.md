# Таблица контекстных триггеров Зинаиды (v2, 10.07.2026)

## Полная таблица

| Первое слово | Контекст | Загружается |
|-------------|----------|-------------|
| `техник` / `техник N` | Инфраструктура, серверы, роутеры | Тех-навыки, SYSTEM_SNAPSHOT, авто-диагностика |
| `дизайн` | Визуал, генерация, LoRA | Replicate, Лера, референсы, design_registry |
| `контент` / `пост` / `smm` | Контент-завод | Фазы, CTA, hooks, RAG, content_factory |
| `план` / `планировщик` | Планирование, задачи | Kanban доска, roadmap, таймеры |
| `писатель` | Написание текстов для проекта | pisatel/00_README.md, стиль Шквальный, чёрный список, RAG/CTA/фазы |
| (ничего) | Без спецконтекста | Только общие правила |

## Формат: {триггер} {проект}

Например:
- `писатель отношения` → грузит `/opt/zinaida/projects/Otnoshenya/pisatel/`
- `дизайн отношения` → грузит `/opt/zinaida/projects/Otnoshenya/design/`
- Если проект не указан → показать список проектов с этой темой

## Структура папки темы

```
/opt/zinaida/projects/{проект}/{тема}/
├── 00_README.md          — выжимка, правила, ссылки
├── 01_*.md               — дополнительные материалы
├── 02_*.md               — дополнительные материалы
├── 03_references.md      — все ссылки и пути
└── 04_approved/          — утверждённые файлы
```
