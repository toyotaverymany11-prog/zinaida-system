# Animated GIF generation — активационные анимации

## Назначение
Генерация узких анимационных GIF-полос для триггеров (маркетинг, глубокое исследование).
ГИФки отображаются в Telegram/чат при активации системы.

## Жёсткие требования (13.07.2026)
- **Размер:** широкий формат, на весь экран — 1920×120
- **Никаких квадратиков.** 800×500 слишком маленькое — Олег ругается.
- Длительность: 2-3 секунды
- FPS: 15 (баланс размера/качества)
- Цвета: тёмный фон (#0a0a0a), циан (#00c8ff), розовый (#ff64c8)
- Текст: "СИСТЕМА АКТИВИРОВАНА" + подтекст

## Скрипт
```bash
python3 /opt/zinaida/design/make_activate_gif.py
```

Выход: `/opt/zinaida/design/system_activate.gif`

## Структура анимации
1. Бегущая сканирующая линия слева направо через весь экран (ядро + свечение)
2. Эхо-линия с отставанием ~20%
3. Матричные полоски-шумы (псевдослучайные, на каждый кадр)
4. Пульсирующие края сверху и снизу
5. Текст по центру, появляется с 30% тайминга, мерцание
6. Подтекст розовым ниже

## Техника (Pillow)
- RGBA для свечений → конверт в RGB при сохранении GIF
- disposal=2 (dispose to background) для чистых кадров
- Кадры генерируются как PNG → собираются в GIF через `first.save(save_all=True, append_images=...)`
- `optimize=True` — сжимает размер

## Питфоллы
- PIL объединяет одинаковые кадры → нужно разное содержимое каждого кадра
- RGBA альфа-каналы не поддерживаются в GIF → конвертить в RGB перед сохранением
- Олег видит GIF в Telegram — размер должен быть <300KB для быстрой загрузки
- формат даты в имени файла кадра — `f'frame_{i:04d}.png'`, а не `{t:04d}` (t — float)
