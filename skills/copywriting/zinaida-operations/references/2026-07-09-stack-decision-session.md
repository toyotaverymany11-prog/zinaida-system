# Сессия выбора стека — 09.07.2026

## Ключевые решения

### 1. Финальный стек
- **Ядро:** SQLite WAL + MCP-сервер + Hermes Cron (на сервере)
- **Визуал:** Hermes Workspace V2 (через Docker в браузере iPad)
- **Голос:** Pipecat (WebRTC → STT → MCP → Hermes → TTS, изолирован от ядра)

### 2. Отложено
- **Claw3D** — 3D офис для AI-агентов, MIT, self-hosted. Когда контент-завод заработает.
- **Генерация видео** — тестировать на консилиуме (Hyperframes не подошёл)

### 3. Отклонено
- LiveKit Agents — заменён на Pipecat
- OpenAI Realtime — только если экстренно
- OpenClaw — не нужен, работаем в Hermes

## Все рассмотренные варианты (8 шт)

1. OpenClaw (382k ⭐) — голос, каналы, cron. Нет аватара.
2. Vellum (100/100) — красивое приложение, голос. Нет Kanban.
3. Hermes Desktop — офиц. приложение Nous Research. Только десктоп.
4. Hermes Workspace (5.6k ⭐) — IDE-подобный веб-интерфейс. Нет голоса.
5. Dograh — open-source Vapi/Retell. Только звонки, нет задач.
6. Open WebUI + Hermes — красивый чат с голосом. Нет Kanban.
7. OpenAI Realtime — идеальный голос. Платный.
8. LiveKit Agents — полный контроль. Требует допиливания.
