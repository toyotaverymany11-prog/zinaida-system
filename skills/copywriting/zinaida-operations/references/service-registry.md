# Реестр сервисов — правила работы

## Как работает
Файл: `/opt/zinaida/shared_memory/service_registry.md`

При старте любого чата AGENTS.md загружается. Правила 7-8 требуют:
1. Перед предложением любого внешнего сервиса — прочитать реестр
2. Если сервис ❌ не работает — не предлагать, назвать альтернативу
3. После пробы нового сервиса — сразу записать результат

## Формат записи
```markdown
### Название сервиса
- **Статус:** ✅ / ❌ / ⏳
- **Проблема:** что не так
- **Дата:** когда пробовали
- **Вывод:** что решили
- **Альтернатива:** что использовать вместо
```

## Зафиксированные нерабочие (по состоянию на 10.07.2026)
- ElevenLabs — блокировка РФ, API ключ не получить
- Linear MCP — OAuth callback на localhost сервера, не работает в РФ
- LiveKit Agents — сложный, глючный, CPU-heavy, Олег отклонил
- Vapi.ai / Retell AI / Bland AI — коммерческие для звонков, не для личного ассистента

## Рабочие альтернативы
- Edge TTS — бесплатно, русский, без ключа, встроен в Hermes
- Silero TTS — локальный, CPU, русский, стоит на порту 7892
- faster-whisper — локальный STT, CPU, бесплатно
- Deepgram STT — ключ есть, $200 кредит
- Telegram voice — работает через Hermes Gateway
- Hermes Kanban — встроенная доска задач
- Todoist — REST API с токеном (не OAuth), есть веб + iPad
