# ПРАВИЛО: НЕ АВТО-ЗАПУСКАТЬ РАБОТУ ПО ПЕРВОМУ СЛОВУ

**Дата:** 11.07.2026
**Контекст:** Олег сказал «писатель отношения» → я немедленно запустила генерацию поста без его команды.

## Суть правила

Первое слово от Олега определяет **тему** (о чём речь), а НЕ является командой к действию.

| Что сказал | Это тема | Команда? |
|---|---|---|
| «писатель отношения» | Контент-завод | ❌ НЕТ, надо спросить |
| «дизайн отношения» | Дизайн | ❌ НЕТ, надо спросить |
| «техник роутер» | Инфраструктура | ✅ ДА, протокол техника |
| «напиши пост про ревность» | Контент-завод | ✅ ДА, есть «напиши» |

## Алгоритм
1. Определить тему по первому слову
2. Если команды нет («пиши», «сделай», «напиши», «запусти») — сказать «Поняла. Какая задача?»
3. Дождаться явной команды
4. Только тогда начинать

## Запрещено
- Запускать скрипты «для подготовки»
- Писать посты без команды
- Говорить «я подготовила, смотри»

## Исключение
«техник» — отдельный протокол с обязательной загрузкой контекста.
