# Тест моделей для обложки с кириллическим текстом (09.07.2026)

## Задача
Создать обложку в стиле Ego Страх (IMG_1288.jpeg) с кириллическим текстом «МЕГАМОЗГ» и лицом Зинаиды.

## Тестированные подходы

### 1. Python/Pillow — ручное наложение текста (провал)
- Попытка: вручную наложить текст Playfair Display через PIL/ImageMagick
- Проблема: шрифт не тот, расположение не по референсу, смотрится как поделка
- Вердикт: **ЗАПРЕЩЕНО**. Творческий дизайн — только через Леру и Replicate.

### 2. Ideogram V3 Quality + Style Reference (частично успешно)
- Модель: `ideogram-ai/ideogram-v3-quality` (дорогая)
- Style Reference: data URI референса Ego Страх
- Результат: стиль похож, текст читается, НО на латинице (MEGAMOZG)
- Время: ~14 сек

### 3. Ideogram V3 Turbo + Style Reference + кириллица (частично)
- Модель: `ideogram-ai/ideogram-v3-turbo` (бесплатно, рекомендуется config.yaml)
- Style Reference: загружен через Files API
- Промпт с кириллицей
- Результат: нижний текст (AI-аналитик) на кириллице ✅, верхний (МЕГАМОЗГ) — латиница ❌
- Время: ~14 сек
- Вердикт: 90-95% точность кириллицы — но верхний текст не гарантирован

### 4. Recraft V3 (провал)
- Модель: `recraft-ai/recraft-v3`
- Стиль: realistic_image
- Результат: текст «ME AMO» + случайные кириллические символы. Полный провал.
- Вердикт: несмотря на запись в config.yaml «лучшая кириллица», не справился.

### 5. Ideogram V3 Turbo v2 — два промпта (частично)
- Версия A (с кавычками-ёлочками): «MEFAMO33Г» — кириллица с искажениями
- Версия B (акцент на кириллицу): «MEГAI MO33F» — мешанина
- Вердикт: кириллица есть, но с искажениями

## Выводы
1. Ни одна AI-модель не делает идеальный кириллический текст + конкретное лицо в одной генерации
2. FLUX+LoRA = лицо Зинаиды, НО нет текста
3. Ideogram V3 Turbo = текст + - -, НО не знает лицо
4. Решение: два прохода — портрет через LoRA, текст через отдельный инструмент/дизайнера
