**Контекст:** Я сказала «GPU нет на сервере». Олег прислал скрины Timeweb Cloud панели с пунктом «Cloud S GPU» (NEW) в меню.

**Правда:** GPU есть — через Timeweb Cloud (timeweb.cloud). Это облачные GPU-инстансы:
- NVIDIA A100, H100, L40S — предположительно
- Доступны из панели управления Timeweb
- Текущий сервер Wild Cygnus (83.217.201.92) — обычный CPU (4 CPU, 8GB RAM, 80GB NVMe)
- GPU надо заказывать отдельно через «Cloud S GPU» в меню

**Что нужно:** зайти в панель Timeweb → Cloud S GPU → посмотреть тарифы и цены. Или запросить API-ключ.

**Урок для будущего:**
- Не утверждать «нет GPU» без проверки Timeweb Cloud панели
- Timeweb Cloud — основной провайдер, GPU там есть как услуга
- Текущий сервер CPU-only, GPU-инстанс нужно создавать отдельно
