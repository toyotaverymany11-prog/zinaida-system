## Требование: проверять ПЕРЕД предложением любого внешнего сервиса

## ❌ НЕ РАБОТАЕТ В РФ (не предлагать)

| Сервис | Причина | Альтернатива |
|--------|---------|-------------|
| ElevenLabs TTS | Заблокирован в РФ, ключ не получить | Edge TTS (встроен, бесплатно) |
| Linear MCP OAuth | Callback на localhost не работает из РФ | Hermes Kanban или Todoist API |
| LiveKit Agents | Сложный, CPU-heavy, Олег не принял | WebSocket + Whisper + Silero/Edge |
| Vapi.ai / Retell AI | США-ориентированные, платные, РФ блоки | Свой WebSocket сервер |

## 🔍 Hermes Studio Voice Dialogue (обнаружено 10.07.2026)

Hermes Studio (hermes-web-ui) уже имеет встроенную поддержку голоса!

Документация: https://github.com/EKKOLearnAI/hermes-studio/blob/main/docs/voice-dialogue.md

**Что работает:**
- Кнопка микрофона в чате (push-to-talk → распознавание → текст в поле ввода)
- TTS ответов агента (через браузер или серверные провайдеры)
- Barge-in: если агент говорит, начав говорить — его аудио останавливается
- Настройки: Settings → Voice (STT/TTS провайдеры)
- Browser STT: использует встроенное распознавание браузера (без ключей!)
- Turn-based: запись → распознавание → редактирование → отправка

**Чего НЕТ (не реализовано в текущей версии):**
- Постоянного прослушивания (wake word)
- Полнодуплексного режима
- Телефонных звонков

**Как настроить:**
1. Открыть Settings → Voice в Hermes Studio
2. Для STT: выбрать Browser Speech Recognition (без ключей) или серверный
3. Для TTS: выбрать Edge TTS (бесплатно) или Browser Speech
4. В чате появится кнопка микрофона
