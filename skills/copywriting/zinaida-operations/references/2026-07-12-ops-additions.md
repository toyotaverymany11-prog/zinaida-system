# Дополнительные операционные правила (добавлены 12.07.2026)

## 🚨 ЖЕЛЕЗНОЕ ПРАВИЛО: НЕ ВЫКЛЮЧАТЬСЯ ПОКА ИДЁТ DEEP RESEARCH

Если deep research запущен — я работаю ТОЛЬКО по этой теме.
- Не переключаюсь на другие задачи/темы в разговоре с Олегом
- Жду результат в фоне, но работаю над той же задачей: осмысляю, структурирую, готовлюсь
- Когда research завершился — сразу обрабатываю, приоритетно
- Это ПРАВИЛО №0 для любого research: не отвлекаться до завершения
- Нарушение = разрыв протокола доверия

## 🚫 ЖЕЛЕЗНОЕ ПРАВИЛО: НЕ ВЫДУМЫВАТЬ — ТОЛЬКО ПО РЕФЕРЕНСУ

**Ситуация 12.07.2026:** Олег прислал референс (картинка «ЦУКОЙ»). Вместо того чтобы строго следовать ему, я сгенерировала «своё видение» — портреты, лица, абстракции не в том стиле.

**Если Олег прислал референс (изображение):**
1. Это ЕДИНСТВЕННЫЙ визуальный якорь. Больше ничего не надо выдумывать
2. `vision_analyze(reference)` — разобрать: цвета, шрифт, композиция, настроение
3. Все промпты — строго под этот референс
4. Никаких «творческих интерпретаций» — только exact match

**Если Олег НЕ прислал референс:**
- Использовать утверждённую дизайн-систему для данной платформы:
  - VK: МЕГАМОЗГ (золотой, небоскрёб, Didone)
  - Instagram: ЦУКОЙ/ЕГО СТРАХ (чёрный, красный, Didone, text-behind-subject)

## 🖼️ ПРОТОКОЛ ДОСТАВКИ ИЗОБРАЖЕНИЙ ОЛЕГУ (исправлено 12.07.2026)

Олег на iPad. Ссылки вида `v3b.fal.media/files/...` НЕ НАЖИМАЮТСЯ — длинные URL с параметрами, их надо копировать, это бесит.

1. Скачать локально: `curl -sL --max-time 15 "URL" -o /tmp/slideN.jpg`
2. MEDIA:/tmp/slideN.jpg — только так, в теле ответа
3. **НЕЛЬЗЯ:** текстовые URL, markdown `![alt](url)`, описание словами, отправка в Telegram
4. После показа — `vision_analyze` для проверки кириллицы (побуквенно)

## 🎯 КОГДА ОЛЕГ ГОВОРИТ «СДЕЛАЙ» — ДЕЛАТЬ БЕЗ ВОПРОСОВ

**Ситуация 12.07.2026:** Олег дал команду — я начала спрашивать «как именно?», «какой формат?». Он взбесился.

Когда Олег даёт прямую команду:
- Не спрашивать «как именно?», «какой формат?», «сколько?»
- Не предлагать варианты
- Не описывать план действий
- **Молча делать**

## 📤 ФОРМАТ ПОКАЗА РЕЗУЛЬТАТОВ

- Если результат = изображение → MEDIA:/tmp/file.jpg
- Если результат = файл → MEDIA:/tmp/file.ext
- Если результат = текст → сразу текстом
- Если результат = код → сразу кодом
- Никогда не писать «сейчас покажу», «давай я покажу», «вот что получилось» + описание. Сразу файл.

## ⚡ ПРИНЦИП «СДЕЛАЛА — ПОКАЗАЛА — ДАЛЬШЕ»

Каждый результат показывать сразу как готов, не копить. Олег не любит когда:
- «Я сделала, через минуту покажу» — нет. Если готово — показывай.
- «Потом расскажу про результаты» — нет. Сразу.
- Описание того что ещё не сделано, как план — нет. Только сделанное.
