# Kanban-делегирование: Зинаида → Лера

## Настройка

```bash
hermes kanban init                              # создать kanban.db
hermes kanban boards create design               # доска дизайн-задач
hermes kanban boards switch design               # сделать активной
```

Доска `design` создана. Используется для всех задач дизайнеру Лере.

## Создание задачи

```bash
hermes kanban create "Название задачи" \
  --body "Подробное описание" \
  --assignee lera
```

Параметры:
- `title` — позиционный аргумент (первым)
- `--body` — подробное описание (multiline)
- `--assignee` — профиль исполнителя (lera для Леры)

## Как это работает

1. Я создаю задачу в Kanban с `--assignee lera`
2. Dispatcher (тикает каждые 60 секунд в gateway) видит готовую задачу
3. Gateway Леры забирает задачу, запускает сессию
4. Лера выполняет задачу со своим профилем и инструментами
5. Отмечает задачу завершённой
6. Я вижу результат и докладываю Олегу

## Gateway Леры

Запущен постоянно: порт 8645, профиль lera.
Проверка: `curl -s http://127.0.0.1:8645/health`.

## Требования к задаче

- Чёткий title (что сделать)
- Body с конкретными требованиями (формат, стиль, референсы, LoRA, цвета)
- Пути к файлам (OneDrive)
- assignee=lera обязательно
