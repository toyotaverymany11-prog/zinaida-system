# Анализ стиля Сатьи Дас (12.07.2026)

**Источник:** Литрес, цитаты из книг, НТВ-передачи
**Для чего:** альтернативная манера письма, когда «шквальный стиль» не работает

## Ключевые черты стиля

1. **Бытовые метафоры из мужского мира**
   - «Женщина — планка для мужчины. На каком уровне висит, такие «спортсмены» и подходят прыгать.»
   - «Мужчина — как Lego. Он всегда продаётся в разобранном состоянии.»
   - «Достаточно ли деталек? Из одного ли они набора? Нет ли ломаных?»

2. **Короткие рубленые фразы** — 3-7 слов, без придаточных предложений
   - «Не рассосётся.»
   - «Твой — не исключение.»
   - «Правда не кусается.»

3. **Прямые риторические вопросы**
   - «Хочешь посмотреть — напиши РАЗБОР.»

4. **Разговорная лексика, просторечия**
   - «мужики», «рассосётся», «деталек»
   
5. **Жёсткая прямота без смягчения**
   - Не «возможно, тебе стоит задуматься», а «готова ли ты открыть глаза»

6. **Отсутствие AI-маркеров**
   - Нет «в контексте», «отметим», «важно понять», «давайте разберёмся»

## Чего у Сатьи НЕТ (и это хорошо)

- Нет описаний действий читательницы («ты берёшь телефон»)
- Нет «это не X, это Y» в любой форме
- Нет «нейробиология говорит» — он использует житейскую мудрость
- Нет воды-связок

## Как использовать в постах Зинаиды

1. Заимствовать метафоры из быта, не из науки
2. Писать рублеными фразами, короткими абзацами
3. Прямые вопросы вместо утверждений
4. Сочетать с научной легитимностью (Зинаида подкованная) — Сатья не даёт науку, мы можем добавить 1 цифру + 1 термин
5. Не копировать, а брать **механику**: бытовая метафора → прямота → вопрос
