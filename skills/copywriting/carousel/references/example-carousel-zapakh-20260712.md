### Ключевые уроки из этой карусели
1. GPT Image 2 на FAL — единственная модель для русского текста любой длины (99%)
2. Для лица Зинаиды — детальное описание (тёмные волосы, ice blue глаза, горбинка носа)
3. FLUX+LoRA — только для лица без текста
4. Стиль «ЦУКОЙ/ЕГО СТРАХ»: чёрный+бордо+белый, text-behind-subject, Didone
5. Сохранять в `inbox/`, не в `SmmFabrika/`
