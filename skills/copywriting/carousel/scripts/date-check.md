# date_check.sh — проверка системной даты

**Путь:** `/opt/zinaida/scripts/date_check.sh`

## Назначение
Единственный source of truth для даты. Выполнять ПЕРЕД любым:
- сохранением файла
- созданием папки
- генерацией
- упоминанием даты в ответе

## Использование
```bash
bash /opt/zinaida/scripts/date_check.sh
```

## Вывод
```
=== ТЕКУЩАЯ ДАТА (единственный источник правды) ===
Системная дата: 2026-07-13 09:27 Monday
Для папок: 2026-07-13
Для подписей: 13.07.2026
✅ Дата валидна (2026 год)
TODAY=2026-07-13
```

## Почему это нужно
13.07.2026: Зинаида сохранила все файлы в папку 2026-07-12 потому что «помнила что вчера было 12-е». Реально было 13-е. Грубейшее нарушение протокола доверия.

## Запрещено
- Брать дату из памяти
- Брать дату из контекста чата
- Брать дату из .env или конфигов
- Наследовать дату из прошлых файлов
