# Cyrillic Verification Protocol (12.07.2026)

> Проблема: AI-модели (Ideogram V3, GPT Image 2) иногда искажают кириллицу.
> Ошибка 12.07: не заметила что текст "ЧУЖОЙ ЗАПАХ" написан с ошибками.

## ДВОЙНАЯ ПРОВЕРКА

После каждой генерации картинки с кириллицей:

### Уровень 1 — vision_analyze
```
Запрос: «Перечисли КАЖДУЮ букву на картинке по порядку. 
Написано "{ожидаемый_текст}" или что-то другое? 
Каждая буква - кириллица или латиница? 
Нет ли латинских букв вместо кириллицы (A/С/Е/Р/Х вместо А/С/Е/Р/Х)?»
```

### Уровень 2 — Mistral Pixtral (если vision вызвал сомнения)
```
Модель: pixtral-12b-2409 через api.mistral.ai
Запрос: "Read the Cyrillic text on this image VERY CAREFULLY.
The correct text should be {...}. List each letter one by one.
Check if any Latin letters got mixed in (P instead of Р, A instead of А, etc.)"
```

### Правило
- Если vision и Pixtral разошлись → скорее всего ошибка → перегенерация
- Tesseract НЕ читает Didone шрифт — не полагаться
- EasyOCR на CPU — медленно, для Didone нестабилен

### Частые ошибки AI при кириллице
| Пишет AI | Должно быть | Модель |
|----------|-------------|--------|
| ЗЛАРАХ | ЗАПАХ | Ideogram V3 (путает А/Р) |
| EГО СТРАХ | ЕГО СТРАХ | Ideogram V3 (E вместо Е) |
| ОСН МОЛЧИТ | ОН МОЛЧИТ | Ideogram V3 (теряет буквы) |
| CHTAYU MUZHIKOV | ЧИТАЮ МУЖИКОВ | Recraft V3 (латиница!) |
| МЕТАМОЗГ | МЕГАМОЗГ | Seedream (путает буквы) |

### Какие модели когда
| Ситуация | Модель | Кириллица |
|----------|--------|-----------|
| Обычная | `fal-ai/gpt-image-2` 🏆 | ✅ 99% |
| Если GPT Image 2 не дал стиль | `fal-ai/ideogram/v3` | ⚠️ 90-95% |
| Только тесты | `fal-ai/flux-2/klein/9b` | ❌ не умеет |

## Важное замечание к GPT Image 2
- Добавляет лишний текст (даты, aesthetic, слоганы) — в промпте указывать:
  `"NO extra text, NO dates, NO watermarks, NO tags"`
- Reference_image_urls для замены лица на этой модели НЕ РАБОТАЮТ — игнорирует
- Для замены лица: только композитинг или FLUX + LoRA
