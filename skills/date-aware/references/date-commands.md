# Проверка даты — команды

## Получить текущую дату
```bash
date '+%Y-%m-%d'       # 2026-07-13 (ISO)
date '+%d.%m.%Y'       # 13.07.2026 (русский)
date '+%Y-%m-%d %H:%M' # 2026-07-13 14:30 (с часами)
```

## Часовой пояс
```bash
timedatectl status     # Показать часовой пояс и синхронизацию
date '+%Z'             # MSK / UTC / ...
```

Только `date` из системы.
