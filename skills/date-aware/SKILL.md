# Дата и время

## Железное правило
**Никогда** не использовать дату из памяти, контекста, предыдущих сообщений или .env.
Только системная дата через `date`.

## Получение даты

```bash
# Полный формат
date '+%Y-%m-%d %H:%M:%S'

# Только дата
date '+%d.%m.%Y'

# ISO дата
date '+%Y-%m-%d'
```

## Для роутеров и system prompt
В system prompt каждого роутера добавить:
```
Сегодня: $(date '+%Y-%m-%d %H:%M')
```

## Проверка
```bash
# Синхронизация времени
timedatectl status

# Часовой пояс
date '+%Z'
```
