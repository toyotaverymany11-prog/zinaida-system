# Маркетинг 2026 — Справочник AI-маркетолога v2.0

## Триггер
Первое слово «маркетинг» в чате. Читается целиком при инициализации.

## Философия
Сдвиг от количества к качеству. Экономика смысла. Human-first > Algorithm-first.
LLM — рендерер, не автор. Нелинейный путь клиента. Намеренное присутствие > ежедневный постинг.

## Ключевые метрики
- **Saves 40%** (3× лайков) — чек-листы, научные факты, фреймворки
- **DM Shares 35%** (3-5×) — стыд, узнавание, идентичность
- **Watch Time 15%** — рваный ритм, маятник +-+, Open Loop
- **Comments 7%, Likes 3%** — игнорировать как цель

## Воронка (3 слоя)
1. **8 соцсетей** — VK, IG, TG, Дзен, OK, Pinterest, MessengerMax, YM. Каждая = native-формат. CTA: «Напиши [название]».
2. **AI-бот** — отвечает за 0.5 сек. Скрипт: «Держи [инструмент] → ссылка. Бесплатно.» Задача: перевод в TG.
3. **Telegram** — полная Зинаида. Автоворонка 3-4 часа: вход → бесплатный результат → арсенал → голос/видео → paywall.

## 5 AI-инструментов
Zero-Token Memory (LTV+35%), Proactive Nudging (возврат 22%), Multimodal Trust (+40% конв.), Deterministic Verifier (OK/EDIT/BLOCK), Micro-Conversion (×3). Проектные оценки.

## 7 слоёв сборки
ДНК → Стиль → Фаза → Attack Layer → RAG → CTA → JSON. Лимит 8000 токенов. Приоритет отрезания: RAG → микро-история → словарь боли → статистика → хук.

## 16 правил стиля
Цифра/факт первым → маятник +-+ → ломай шаблон → метафоры из быта → глаголы → абзац 1-2 строки → грязная правда → повелительное → Open Loop → эмодзи-пуля → спорт → дефицит → честность → стрелка в конце.

## 152-ФЗ
Дисклеймер в тоне Зинаиды. Алгоспик в config.json, bulk-скрипт. BLOCK → ADMIN_CHAT_ID.

## Целевые показатели
Пост→TG CTR ≥ 5%, TG→paywall 15-25%, free→paid ≥ 8%, retention M2 ≥ 60%, LTV/CAC 3:1-5:1.
