# Система «Зинаида» — AI-ассистент с памятью и самообучаемостью

Полноценная AI-система для сервера. Агент, который помнит всё, учится сам, реагирует на команды первым словом и умеет глубоко исследовать любую тему.

---

## ⚡ Быстрый старт

**Одна команда для установки на сервер:**
```bash
sudo apt update -y && sudo apt install git -y && git clone https://github.com/toyotaverymany11-prog/zinaida-system.git /opt/zinaida && cd /opt/zinaida && sudo bash install.sh
```

После завершения открыть в браузере: **http://IP_СЕРВЕРА:5000**

Установщик сделает всё сам:
- Установит зависимости (Python, Node.js, Hermes CLI)
- Настроит systemd-сервисы
- Запустит роутеры
- Запустит Hermes Web UI
- Создаст агента

## Архитектура (7 слоёв)

```
DNA → Стиль → Фаза → Attack Layer → RAG → CTA → JSON
```

Каждый пост/ответ собирается из атомарных деталей как конструктор.

## Самообучаемость

- Mem0 — автоматическая память с доверием к фактам
- Обратная связь — агент учится на своих ошибках
- Петля обучения: генерация → публикация → метрики → адаптация
