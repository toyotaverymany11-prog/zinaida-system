## ⚡ Быстрый старт

**Одна команда — и система на сервере:**

```bash
sudo apt update -y && sudo apt install git -y && git clone https://github.com/toyotaverymany11-prog/zinaida-system.git /opt/zinaida && cd /opt/zinaida && sudo bash install.sh
```

После установки — просто пиши в чат. Первое слово = команда.

## 🚀 Установка

```bash
git clone https://github.com/toyotaverymany11-prog/zinaida-system.git /opt/zinaida && bash /opt/zinaida/install.sh
```

Установщик сам:
1. Поставит Docker, Node.js, Python-зависимости
2. Скопирует навыки и протоколы
3. Настроит роутеры
4. Создаст systemd-сервисы
5. Установит и запустит Hermes CLI + Web UI
6. Запросит API-ключи
